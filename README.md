# Goalkeeper Trainer Demo

Demo PWA per allenamento portieri.

## Come provarla su Windows

Metodo semplice:
1. Estrai lo ZIP.
2. Apri `index.html` con Chrome o Edge.

Metodo migliore:
1. Installa VS Code.
2. Installa estensione "Live Server".
3. Apri la cartella.
4. Tasto destro su `index.html` > "Open with Live Server".

## Come pubblicarla

Carica questi file in un repository GitHub e pubblica con GitHub Pages, Netlify o Cloudflare Pages.

Su iPhone:
1. Apri il link da Safari.
2. Condividi.
3. Aggiungi alla schermata Home.
4. Attiva "Apri come app web", se disponibile.
